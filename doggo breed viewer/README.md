# DOGGO BREED VIEWER

A project using dog.ceo API, built using the tutorial provided by [MayanWolfe](https://www.youtube.com/watch?v=rHnC-7h6338&t=16014s)


## TECH STACK

- HTML / CSS / JavaScript;
- API: [Dog Ceo](https://dog.ceo/dog-api/)


## FEATURES

- Use a dropdown menu to choose a dog breed,
- The application fetches the image from the dogceo API and updates it,
- It displays a random name, age, likes, dislikes and a fun fact
